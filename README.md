# Control

Control 是基于 [UI-TARS](https://github.com/bytedance/UI-TARS) 修改而来的项目，用于向原项目致敬。

本项目在原项目许可证基础上进行修改，沿用 `Apache License Version 2.0, January 2004`。

## 启动命令

### 安装依赖

```bash
pnpm install
```

### 开发启动

```bash
pnpm --filter wu-ji-desktop dev
```

### 编译构建

```bash
pnpm --filter wu-ji-desktop build
```
