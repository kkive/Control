# Control

Control 是基于 [UI-TARS](https://github.com/bytedance/UI-TARS) 修改而来的项目，用于向原项目致敬。

最新版本是0.2.9，现在支持对接个人微信，实现个人微信操作电脑了

## 启动命令

### 安装依赖

```bash
pnpm install
```

### 开发启动

```bash
pnpm --filter control-desktop dev
```

### 编译构建

```bash
pnpm --filter control-desktop build
```
